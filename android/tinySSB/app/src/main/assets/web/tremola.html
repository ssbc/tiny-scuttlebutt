Welcome to Quiz Master!

About the Game

Authors: Anoozh, Prabavan and Jakob

This Game is made for fun and educational purposes. We ask you to create a Quiz Set, which is solvable and reasonable. If you think a Question Set is not reasonably solvable, then please press the "dislike"-button

How does it work?

"Ranking": Look at the current Rank-List

"Enter Game": If you want to answer some questions, just press on an open Game.

There are two different Question-types:

"SCQ" single choice Question

"MCQ" multiple choice Question

In case you thought, that the questions were not reasonable, then press the "dislike"-button. After two dislikes, the responsible player will loose two points.

"Create Game: " If you want to create you own Game, press on "New Question Set" or if you want to look at the Question Sets, that you created, press on "List Question Set".

You can create two different Questions:

single choice Questions (SCQ) and multiple choice Questions (MCQ)

The choice depends on the type Box, you selected. If you choose SCQ, you can only select one correct Solution.
